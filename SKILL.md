# A股私募基金行业分析 Skill
本skill用于指导如何规范、完整地生成一份A股私募行业分析参考Markdown文件。

触发条件（出现以下任意关键词/情境时使用本skill）：
- 私募行业分析、私募行业研究、私募市场报告
- 量化私募 AUM、量化梯队、量化规模
- 百亿私募、私募业绩、指数增强、超额收益
- A股私募监管、中基协数据、私募基金市场综述
- 用户要求生成”私募行业参考”、“私募分析框架”

## 输出规范
- **格式**：标准 Markdown，含目录锚点、表格、时间轴代码块
- **文件命名**：`中国私募行业分析参考_YYYY.md`（YYYY为数据截止年份）
- **长度目标**：约 400–600 行，覆盖七大模块，不可随意裁减

## 工作流程

### Step 1：获取最新数据（必须执行）

在生成报告前，**必须**通过联网搜索抓取以下核心数据，不得依赖训练数据（行业规模和业绩数字时效性强，过期即失真）：

|查询目标     |推荐搜索词                                      |
|---------|-------------------------------------------|
|行业总规模    |`中基协 私募基金 存续规模 [最新年月]`                     |
|量化私募AUM图谱|`量化投资与机器学习 AUM图谱 [最新季度]` 或 `QIML 量化私募 管理规模`|
|头部机构梯队   |`量化四大天王 AUM [年份]` / `幻方 九坤 明汯 衍复 规模`       |
|业绩数据     |`百亿私募 年度业绩 [年份] 私募排排网`                     |
|监管新规     |`私募基金 监管新规 [年份]` / `中基协 备案指引 [年份]`         |
|行业趋势     |`量化私募 超额收益 容量 [年份]`                        |

**数据源优先级**：中基协官方月报 > 私募排排网 > QIML公众号季报 > 21经济报道/上海证券报 > 其他

### Step 2：按模块结构生成报告

严格按照以下**七大模块**组织内容，每个模块均不可省略：

## 报告结构模板

在生成文件时，完整复现以下结构：

```
# A股私募行业深度分析参考
**数据截止：[具体年月] / 报告生成：[当前日期]**
数据来源：[列举本次实际使用的信息来源]

## 目录
[含7个模块的锚点链接]
```

### 模块一：行业整体规模

必须包含：

- **总量概览表**：存续管理人数、基金数、总规模、同比增幅
- **三大类型拆分表**：私募证券 / 私募股权 / 创业投资（管理人数、基金数、规模）
- **地域分布**：管理人数量前6辖区（占比%） + 管理规模前6辖区（金额）
- 简短结构性点评（2–3行，说明各类型占比特征）

### 模块二：私募证券基金专项

必须包含：

- 存续证券基金规模及里程碑
- 当年全年备案产品数量及同比增速
- 百亿私募机构总数、量化与主观机构数量对比
- 行业集中度现象描述（头部虹吸、小微消亡）

### 模块三：量化私募AUM格局与梯队

> 本模块数据主要依赖 **QIML（量化投资与机器学习）公众号**季度AUM图谱，每季度更新，搜索时注明最新季度。

必须包含：

- **行业总规模时间表**（至少3个时间节点，含私募量化+公募量化）
- **头部梯队分层表**（第一/二/三梯队，含代表机构、规模区间、策略特点）
- **格局演变时间轴**（从2018年量化元年至当前，用 ````` 代码块排版）
- 灵均等重要机构的升降分析（动态变化是重点）

梯队划分参考框架（规模数字需根据最新数据更新）：

|梯队  |规模区间    |代表机构                |
|----|--------|--------------------|
|第一梯队|[最新数据填写]|幻方、明汯、九坤、衍复（或当期实际情况）|
|第二梯队|[最新数据填写]|世纪前沿等               |
|第三梯队|[最新数据填写]|灵均、诚奇、黑翼等           |
|百亿其他|100亿以上  |茂源、蒙玺、龙旗、稳博等        |

### 模块四：业绩表现

必须包含：

- **全市场私募证券业绩汇总表**（按策略分类：股票/多资产/期货衍生品/FOF/债券，含产品数、平均收益、正收益占比）
- **百亿私募业绩细分表**（量化 / 主观 / 混合，含机构数、平均收益、正收益占比、代表机构）
- **量化细分策略业绩表**（中证500指增、中证1000指增、量化选股等，含超额收益数据）
- 与上一年业绩对比（至少简述上年同期情况作为参照基准）

### 模块五：监管动态

必须包含：

- **年度监管主线**（“X+X”模式总结，如”促发展+强规范”），用表格呈现各方向核心举措
- **主要新规梳理表**（时间 | 法规名称 | 核心要点，覆盖当年主要政策）
- **监管检查重点清单**（编号列表，5条以上）

### 模块六：行业趋势与风险提示

必须包含：

- **主要趋势**：至少4条，每条含标题（**加粗**）+ 2–3行说明
- **核心风险提示表**（风险类型 | 具体内容 | 风险等级），至少5类风险，等级用”⚠️ 高度关注 / 中等 / 低-中”标注

### 模块七：数据来源与更新指引

必须包含：

- **权威数据源汇总表**（数据类型 | 来源 | 更新频率）
- **定期跟踪清单**（月度/季度/半年度/年度，明确建议跟踪动作）
- **免责声明**（私募不得公开宣传业绩，数据仅供内部研究参考）
- 文件版本号、生成时间、下次建议更新时间

## 格式规范细节

- 所有数据表格用标准 Markdown 表格（`|---|`）
- 时间轴用三个反引号代码块排版，保持视觉整洁
- 数据单位统一（亿元/万亿元，明确标注）
- 增速/对比数据用**加粗**标出关键数字
- 风险等级加粗标注增强可读性
- 文件末尾标注版本信息一行

## 质量检查清单

生成完毕后，核验以下项目：

- [ ] 七大模块均已覆盖，无遗漏
- [ ] 所有规模数字来自搜索结果（非记忆数据）
- [ ] 量化梯队数据注明来源为 QIML 或其他具体来源
- [ ] 业绩数据包含量化与主观的对比
- [ ] 监管新规时间精确到月份
- [ ] 风险提示表包含”容量衰减”和”市场风格切换”两类
- [ ] 数据源表中包含中基协、QIML、私募排排网三个核心来源
- [ ] 文件末尾有版本号和下次更新建议时间
